Project 1: I Like To Move It

"I Like To Move It" is a full-stack web application designed to help immigrant children in Melbourne find parks and stay active. Built with React and Bootstrap for the frontend and Python Flask for the backend, it integrates Google Maps API for navigation and OpenWeather API for real-time weather updates. The backend processes user interactions, manages game activities using TensorFlow Lite for image recognition, and stores data in an SQLite database. The interactive interface offers personalized recommendations, safety ratings, and engaging challenges to encourage outdoor activities among children.

Project 2: CycleBoost

CycleBoost is a web-based application designed to promote cycling as a safer and more sustainable commuting option in Melbourne. Built with Dash-Leaflet for interactive map visualizations and powered by Python and SQLite, the application integrates Google APIs for real-time route planning and directions. Key features include a choropleth map displaying bicycle accident hotspots, a bar chart analyzing accident severity, and a route planner that factors in designated bike lanes, elevation changes, and difficulty levels. By combining historical safety data with interactive mapping, CycleBoost helps cyclists make informed decisions for safer and more efficient journeys.

Project 3: How safe are our children once they leave school, in the city of Casey?

How Safe Are Our Children Once They Leave School in the City of Casey? is an interactive data visualization project analyzing school locations and road accident patterns in the City of Casey. It visually maps student distributions and accident clusters to assess potential safety risks for children commuting to and from school. Data preprocessing was performed using R in R Studio, involving cleaning, formatting, filtering, and merging operations before exporting to CSV. The visualization was developed using D3.js within a JavaScript-driven HTML document, incorporating Turf.js for geospatial analysis and a customized D3-based slider for enhanced interactivity. This project provides valuable insights to inform safety measures and improve student protection on city roads.

Project 4: Bakery Sales Android App

Sweet Cakes is a Kotlin-based mobile application that enables users to browse and order cakes from a bakery with an intuitive and visually engaging interface. The app features secure Google Sign-In using Credential Manager, a structured user authentication system, and a profile management screen. Users can explore a variety of cakes, view detailed product descriptions with calorie estimates from an external source, and add selected items to their cart. The app's product and order data are efficiently managed using Room database, ensuring smooth data persistence. Built with modern Android components like Room, Retrofit, LazyColumn, and Coil, Sweet Cakes delivers a seamless and structured user experience.

Project 5: Medical Imaging Service Web Portal

Medical Imaging Service Web Portal is a full-stack web application built with ASP.NET (MVC), C#, Razor markup, and Bootstrap, featuring a MS SQL database backend with role-based authentication for secure access control. The system enables patients to create accounts, manage appointments, and submit service ratings, while practitioners and administrators handle bookings, user management, and communication. Administrators can view aggregated practitioner ratings, generate reports with data export (CSV/PDF), and manage schedules and center closures. The platform integrates Google Maps API for location services, data visualization with interactive charts (scatter plots and tables), and a bulk email system with attachments, optimizing operational efficiency.

Project 6: High-Fidelity Mobile App Prototype

"Plate Pledge" is a mobile app prototype designed in Figma to streamline the food donation process by helping donors locate nearby donation centers, access guidelines, explore charity events, and track their contributions. Built with a user-centered design approach, the prototype features interactive elements, including a home screen with a clickable map, a reminder system for donation scheduling, and a navigation menu for additional options.
